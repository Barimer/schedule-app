# 📅 Focus & Flow: 1-4-7-14 복습 스케줄러

공부 습관 형성과 체계적인 복습을 돕는 현대적인 미니멀리즘 스케줄러입니다. 에빙하우스 망각 곡선에 기반한 **1-4-7-14 복습 알고리즘**을 핵심으로 하며, 미룬 과제를 추적하는 **부채 시스템**과 일상 루틴을 관리하는 **습관 탭**을 제공합니다.

---

## 🚀 주요 기능

### 1. 1-4-7-14 복습 알고리즘
- **신규 학습:** 매일 정해진 새로운 진도를 나갑니다.
- **누적 복습:** 신규 학습일로부터 1일, 3일, 6일, 13일 후에 복습 일정이 자동으로 배치됩니다.
- **학습 데이터:** `plan.md`에 정의된 56일간의 실제 의학/보험 이론 학습 계획이 모두 반영되어 있습니다.

### 2. 미루기(Debt) 시스템
- 오늘 완료하지 못한 과제는 내일 일정 상단에 **'Delayed Tasks'** 섹션으로 자동 이월됩니다.
- 미룬 과제에는 빨간색 **DEBT** 태그가 부여되어 시각적인 피드백을 제공합니다.

### 3. 습관(Habit) 관리
- 공부 외에 매일 반복하고 싶은 일상 루틴을 추가하고 관리할 수 있습니다.
- 간단한 체크 기능과 삭제 기능을 지원합니다.

### 4. 시각적 로드맵
- **캘린더 모달:** 56일 전체 여정 중 현재 위치와 과거 달성률(🌟)을 그리드 형태로 확인합니다.
- **전체 일정(Full Roadmap):** 주차별로 그룹화된 상세 학습 내용을 리스트 형태로 훑어볼 수 있습니다.

---

## 🛠️ 기술 스택
- **Frontend:** React (Vite), TailwindCSS
- **State & Storage:** React Hooks (useState, useMemo), LocalStorage (영구 저장)
- **Design:** Modern Minimalist 'Focus & Flow' (Inspired by Google Stitch)
